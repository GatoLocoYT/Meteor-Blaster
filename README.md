# ☄️ Meteor Blaster

A simple 2D arcade space shooter developed with **Python** and **Pygame**.

Originally created as a small personal project to practice game development and have a quick game to play during short breaks.

## 🎮 Features

- 🚀 Control a spaceship and destroy incoming meteors.
- ☄️ Random meteor generation.
- 🎯 Score and High Score system.
- ❤️ Three-life gameplay.
- 🎵 Background music and shooting sound effects.
- 🔄 Restart option after Game Over.

---

## 🕹️ Controls

| Key | Action |
|------|--------|
| ← / → | Move the spaceship |
| Space | Shoot |
| Y | Restart after Game Over |
| N | Exit the game |

## 🚀 Running from Source

Clone the repository:

```bash
git clone https://github.com/your-username/Meteor_Blaster.git
cd Meteor_Blaster
```

Install Pygame:

```bash
pip install pygame
```

Run the game:

```bash
python Meteor_Blaster.py
```
